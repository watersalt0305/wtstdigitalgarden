## 第一次相遇（天灾前，三个月前）
- 调查队追捕"非法者"（没有在存档名的伦）
- 明栖被临时抽调（人手不够，需要熟悉药剂的人）
- 伊查负责追踪
- 在界外森林边缘堵住伦
- 关键时刻：明栖举枪，伦转身，两人对视
- 明栖看见伦的眼睛——空的、没有焦点
- 明栖脑中"裂开"，涌出白色走廊、编号（IA-1, IA-2...）的碎片记忆
- **明栖没有开枪**，伦逃脱
- 伦那边也触发了记忆碎片（"IA-1，请配合检测"）
- 任务失败，明栖申请回研发岗

## 明栖的转变线
1. 第一次相遇 → 没开枪 → 裂缝炸开 → 回研发岗躲避
2. 但裂缝没有愈合，反而在扩大
3. 主动找到伦（想要答案）
4. 跟伦跑了 → 某些记忆回来 → 意识到自己的记忆是**被人动过的**
5. 遇见芍药花堂 → 了解到精神控制机器的存在
6. 把两件事连起来：**是这个东西删了我的记忆**
7. 恐惧 + 反感 → 加入这次行动（对她来说是私人的）

## 天灾期间
伦的创伤线：
- 伦被方的Zero+Night碎片控制（短暂消除记忆）
- 在碎片控制下杀死了辛达西斯（完全没有意识，像空壳行动）
- 控制解除后突然清醒，看到自己手上的血
- 意识到自己做了什么→成为重大创伤点/PTSD

伊查的目击：
- 伊查追踪时与队伍暂时分离
- 返回时看到现场：辛达西斯倒在血泊中，伦站在旁边手上沾满血迹
- 两人有一瞬间对视
- 伊查成为唯一目击者，但只看到结果，不知道伦是被控制的

调查队其他成员：
- 北瑞和罗卡德死于天灾
- 只有伊查存活（因人类本质，但他不知道原因）

**审讯细节：**

- **毫无预兆抬手，枪口抵上伦额头**

  - "六年前。SEED火灾。十四个实验体从ZB区逃跑。"

  - "你从河里捞上来的时候体温31度。瞳孔对光反射消失。"
  - "西鲁特在你身上做了四十分钟心肺复苏。"
- **"没死透"**——准确、冷、不带感情（不是"活下来"不是"幸存"）

- **"什么都不知道就来了"**——唯一泄露的恼怒只在"什么"的气音

## 伦逃跑事件

### 救援与混乱（关键转折）
1. 不明身份的人冲入STEM
2. 灯儿启动Ex.Ruler模式2
3. 所有人被困住——**伦发现自己能动**
4. **这时候**伦才意识到自己不受限制
5. 伦开始跑
6. **这时候**Crovet标记异常，灯儿/卡卡才知道有两种模式
7. **这时候**灯儿意识到关键问题——

### 灯儿意识到的不是"伦是人类"
- ZEAL参与者都知道伦是人类
- 灯儿意识到的是：**碎片不能影响人类**
- 进一步推导：碎片能影响我们 → 碎片是不是只能影响碎片？
- **那我们是什么？我们的本质是什么？**
- 城区本来没有植物（自然植物会影响碎片）
- 动物 + 碎片 = 我们？→ 我们是被碎片改造的动物？
- **灯儿是最先意识到这个本质的人**

### 灯儿的选择
- 放走伦
- 让伦去发现、去明白
- 同时灯儿在思考更大的问题：
  - 自然力量/规则会压制规则外的东西
  - 精神控制机器——它真的导向坏的结局吗？
  - 界外森林那帮人主张让自然力量入侵、瓦解碎片
  - **哪个造成的混乱更大？**
  - **这个选择交给谁？**

### 走廊尽头
- 伦跑到死路，门需要C级权限
- 卡卡单独出现，告知：模式2对伦无效，Crovet标记了异常
- "你不应该能动。但你动了。"
- 卡卡主动开门 + 告知逃跑路线 + 去拖延灯儿

### 灯儿的反应
- 卡卡谎称伦往东边跑
- 灯儿看着追踪信号往西移动，什么都没说
- 灯儿下令：关闭追踪，理由"维护"
- 灯儿下令：IA-1档案加密等级提升到S
- **灯儿也在放水**——她需要伦活着、去发现真相
